\doxysubsection{Detailed Description}
Class to parse the input and call the appropriate functions

This class provides functionality to parse the input and call the appropriate functions. 

This file contains the implementation of the \doxylink{classgl__input}{gl\+\_\+input} class. The \doxylink{classgl__input}{gl\+\_\+input} class represents the input to the library system. This class provides functionality to parse the input and call the corresponding function.
