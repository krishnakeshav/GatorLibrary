\doxysubsection{Detailed Description}
Header file for the \doxylink{class_r_b_tree}{RBTree} class and \doxylink{class_rbt_node}{Rbt\+Node} class. 

Represents a red-\/black tree data structure.

This file defines the \doxylink{class_r_b_tree}{RBTree} class and \doxylink{class_rbt_node}{Rbt\+Node} class, which implement a red-\/black tree data structure. The \doxylink{class_r_b_tree}{RBTree} class provides methods for adding, removing, and finding nodes in the red-\/black tree. The \doxylink{class_rbt_node}{Rbt\+Node} class represents a node in the red-\/black tree and stores a key-\/value pair.

The \doxylink{class_r_b_tree}{RBTree} class represents a red-\/black tree data structure. It provides methods for adding, removing, and finding nodes in the red-\/black tree.
