\doxysubsection{Detailed Description}
Represents a book in the library. 

This class provides functionality to manage reservations, borrowings, and information about a book. It contains member variables to store the book\textquotesingle{}s ID, name, author, availability status, and borrower. The class also includes a priority queue for managing reservations. \begin{DoxySeeAlso}{See also}
